### Using AI Suggestions

BriefLi includes AI-powered slug suggestions to help you create memorable links.

#### How to Use

1. Enter your **Destination URL** first
2. Optionally add a **Description**
3. Click the **âœ¨ AI Suggest** button
4. Wait for AI to generate 5 creative suggestions
5. Click any suggestion to use it as your slug

#### Tips for Better Suggestions

- Add a descriptive description for more relevant suggestions
- Longer URLs give AI more context to work with
- You can always edit the suggested slug
